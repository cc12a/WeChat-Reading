# wxRead插件


</div>

## 使用方法

- 安装完后，侧边栏会出现读书icon ，点击即可进入微信读书页面
- 点击登录，进入我的书架即可阅读图书
- 窗口可以左右拖动调整大小

## 快捷键
- 打开插件
</br>windows：ctrl + r </br>
mac：cmd + r

- 隐藏插件</br>
windows：ctrl + l</br>
mac：cmd + l

## 事项注意
本插件只能阅读我的书架的图书，对于未加入到书架的图书不能进行阅读
